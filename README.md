# International Breweries Dashboard

An interactive Excel dashboard analyzing international breweries data. The dashboard includes:

- Pivot tables summarizing sales, revenue, and brewery performance
- Charts visualizing trends and comparisons
- A comprehensive dashboard view for quick insights

## File

- `International Breweries.xlsx` – contains all pivot tables, charts, and the dashboard in one file.

## How to Use

1. Open `International Breweries.xlsx` in Microsoft Excel (2016 or later recommended).
2. Navigate through the different sheets:
   - **Data** – raw data used for analysis
   - **Pivot Tables** – summarized insights
   - **Charts** – visualizations of key metrics
   - **Dashboard** – interactive overview of all insights
3. Interact with slicers and filters in the dashboard to explore different dimensions.
